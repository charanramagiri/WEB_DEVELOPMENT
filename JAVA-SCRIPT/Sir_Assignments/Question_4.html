JavaScript Variable Scope

1. Global Scope

Declared outside functions or blocks. Accessible anywhere in the code.

var globalVar = "I am global";

function showGlobal() {
  console.log(globalVar); // Accessible
}

showGlobal();

Output: I am global

2. Function (Local) Scope

Declared inside a function. Accessible only within that function.

function localScope() {
  let localVar = "I am local";
  console.log(localVar); // Accessible
}

localScope();
// console.log(localVar); // Error

Output: I am local

3. Block Scope

Variables declared with let and const are block scoped.

{
  let blockVar = "Block scoped";
  console.log(blockVar); // Accessible inside block
}
// console.log(blockVar); // Error

Output: Block scoped

4. Variable Shadowing

A local variable with the same name as a global one will shadow it within its scope.

let name = "Global";

function shadowExample() {
  let name = "Local"; // Shadows the global variable
  console.log(name);
}

shadowExample();
console.log(name);

Output:
Local
Global
